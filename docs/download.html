설치 방법

  1. 위 QR 코드를 스캔하거나 버튼 클릭으로 APK 다운로드
  2. Android 설정 → 보안 → 출처를 알 수 없는 앱 허용
  3. 다운로드된 APK 파일을 눌러 설치
  4. 앱 실행 후 BLE 디바이스 연결하여 SDK 검증 시작
